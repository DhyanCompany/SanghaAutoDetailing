## 📞 **Support & Troubleshooting**

### Email Not Sending?
- Check you updated the Public Key in `js/script.js`
- Verify template name is `template_sangha_booking`
- Check browser console for errors (F12 → Console)
- Verify Gmail is connected in EmailJS

### Form Not Working?
- Check browser console for JavaScript errors
- Ensure all fields are filled
- Verify field IDs match form names

### Animation Issues?
- Check that `css/style.css` is properly linked
- Clear browser cache (Ctrl+Shift+Delete)
- Try different browser to test
